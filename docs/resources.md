## 集成平台

### Composio

[Composio](https://composio.dev) 为 1000+ SaaS 应用提供预构建的 Claude Skills。

- [工具包列表](https://composio.dev/toolkits) - 查看所有支持的应用
- [平台文档](https://docs.composio.dev) - 详细集成指南

支持的应用类别：
- **CRM 与销售**: Salesforce, HubSpot, Pipedrive, Close, Zoho CRM
- **项目管理**: Jira, Linear, Notion, Asana, Trello, ClickUp
- **沟通**: Slack, Discord, Microsoft Teams, Telegram
- **代码托管**: GitHub, GitLab, Bitbucket
- **存储**: Google Drive, Dropbox, OneDrive, Box
- **社交媒体**: Twitter/X, LinkedIn, Instagram, Reddit
- **邮件**: Gmail, Outlook, SendGrid, Postmark
- **数据库**: PostgreSQL, MongoDB, Supabase
