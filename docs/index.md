## What are Claude Skills? | 什么是 Claude Skills？

**English**: Claude Skills are customizable workflows that teach Claude to perform specific tasks tailored to your unique needs. Skills enable Claude to execute tasks in a repeatable, standardized manner across all Claude platforms.

**中文**: Claude Skills 是可定制的工作流程，教会 Claude 根据你的独特需求执行特定任务。Skills 让 Claude 能够在所有 Claude 平台上以可重复、标准化的方式执行任务。

## Quick Start | 快速开始

### In Claude.ai | 在 Claude.ai 中使用

1. Click the skills icon (🧩) in the chat interface
2. Add skills from the marketplace or upload custom skills
3. Claude automatically activates relevant skills based on your task

1. 点击聊天界面中的技能图标（🧩）
2. 从市场添加技能或上传自定义技能
3. Claude 根据你的任务自动激活相关技能

### In Claude Code | 在 Claude Code 中使用

```bash
# 1. Place skills in the skills directory
mkdir -p ~/.config/claude-code/skills/
cp -r skill-name ~/.config/claude-code/skills/

# 2. Start Claude Code
claude
```

Skills load automatically and activate when relevant.

技能自动加载并在相关时激活。
