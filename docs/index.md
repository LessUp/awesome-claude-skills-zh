# Awesome Claude Skills（中文版）

> 本项目是 [awesome-claude-skills](https://github.com/ComposioQ/awesome-claude-skills) 的中文翻译版本

精选的实用 Claude Skills 列表，用于提升 Claude.ai、Claude Code 和 Claude API 的生产力。

## 快速开始

### 在 Claude.ai 中使用技能

1. 点击聊天界面中的技能图标
2. 从市场添加技能或上传自定义技能
3. Claude 根据你的任务自动激活相关技能

### 在 Claude Code 中使用技能

1. 将技能放在 `~/.config/claude-code/skills/`
2. 启动 Claude Code
3. 技能自动加载并在相关时激活

## 技能分类

- [文档处理](../#文档处理) - Word、PDF、PPT、Excel
- [开发与代码工具](../#开发与代码工具) - MCP Builder、Webapp Testing 等
- [数据与分析](../#数据与分析) - CSV、PostgreSQL 等
- [商业与营销](../#商业与营销) - 品牌指南、线索研究等
- [沟通与写作](../#沟通与写作) - 内容研究、会议分析等
- [创意与媒体](../#创意与媒体) - 图像增强、GIF 创建等
- [生产力与组织](../#生产力与组织) - 文件组织、简历生成等
