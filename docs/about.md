# 关于本项目

**Awesome Claude Skills 中文版** 是 [awesome-claude-skills](https://github.com/ComposioQ/awesome-claude-skills) 的中文翻译版本。

---

## 翻译目的

Claude Skills 是一个快速增长的生态系统，但大部分资源都是英文的。本翻译旨在：

1. **降低语言障碍** - 让更多中文开发者能够使用 Claude Skills
2. **推广技术** - 帮助国内开发者了解和掌握这一强大工具
3. **社区建设** - 聚集中文 Claude 用户，共同交流和学习

## 关于 Claude Skills

Claude Skills 是 Anthropic 推出的可定制工作流程系统，允许用户教会 Claude 执行特定任务。

### 核心特点

- **跨平台** - 一次创建，可在 Claude.ai、Claude Code 和 API 中使用
- **模块化** - 每个技能自包含，易于分享和复用
- **可扩展** - 通过脚本、模板和参考文件增强能力
- **智能激活** - Claude 自动判断何时使用相关技能

### 适用场景

- 📄 文档处理（Word、PDF、Excel、PPT）
- 💻 软件开发（代码生成、测试、调试）
- 📊 数据分析（CSV、数据库、可视化）
- 🎨 创意媒体（图像、视频、设计）
- 📈 商业营销（竞品分析、客户研究）
- ⚡ 自动化流程（邮件、通知、任务管理）
