# 抽奖获胜者选择器

此技能从列表、电子表格或 Google Sheets 中随机选择抽奖和竞赛的获胜者。

## 何时使用此技能

- 运行社交媒体抽奖活动
- 在活动中选择抽奖获胜者
- 随机选择调查或测试的参与者
- 从竞赛提交中选择获胜者
- 公平分配有限名额或资源
- 随机团队分配

## 此技能的功能

1. **随机选择**：使用密码学随机选择
2. **多种来源**：适用于 CSV、Excel、Google Sheets 或纯列表
3. **多个获胜者**：可以选择一个或多个获胜者
4. **防止重复**：确保同一人不能赢两次
5. **透明结果**：清晰显示选择过程
6. **获胜者详情**：显示关于获胜者的所有相关信息

## 如何使用

### 来自 Google Sheets

```
从这个 Google Sheet 中随机选择一行来选择抽奖活动的获胜者：[Sheet URL]
```

### 来自本地文件

```
从 entries.csv 中随机选择 3 个获胜者
```

### 来自列表

```
从这个列表中随机选择一个获胜者：
- Alice (alice@email.com)
- Bob (bob@email.com)
- Carol (carol@email.com)
...
```

### 多个获胜者

```
从 contest-entries.xlsx 中随机选择 5 个获胜者，
确保没有重复
```

## 功能

### 公平选择
- 使用安全的随机数生成
- 无偏见或模式
- 透明过程
- 可重复带种子（用于验证）

### 排除
```
排除之前的获胜者 Alice、Bob、Carol 随机选择一个获胜者
```

### 加权选择
```
基于"entries"列选择带加权概率的获胜者（1 个条目 = 1 张票）
```

### 候补获胜者
```
从列表中选择 1 个获胜者和 3 个候补获胜者
```

## 示例工作流

### 社交媒体抽奖
1. 从 Google Form 导出条目到 Sheets
2. "从 [Sheet URL] 中随机选择一个获胜者"
3. 验证获胜者详情
4. 公开宣布并带时间戳

### 活动抽奖
1. 创建参与者姓名和邮箱的 CSV
2. "从 attendees.csv 中随机选择 10 个获胜者"
3. 导出获胜者列表
4. 直接邮件通知获胜者

### 团队分配
1. 有参与者列表
2. "将此列表随机分成 4 个相等的团队"
3. 审查分配
4. 分享团队名单

## 提示

- **记录过程**：保存时间戳和方法
- **公开宣布**：分享选择详情以增加透明度
- **检查资格**：验证获胜者符合竞赛规则
- **有备份**：选择候补获胜者以防获胜者不符合资格
- **导出结果**：保存获胜者列表用于记录

## 隐私与公平
