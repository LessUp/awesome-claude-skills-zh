# Composio 技能集合

此目录包含通过 [Composio (Rube MCP)](https://composio.dev) 集成的 78+ SaaS 应用的预构建工作流技能。

- Box、Dropbox、Google Drive、OneDrive 等

### 日历与调度
- Cal.com、Calendly、Google Calendar、Outlook Calendar 等

### 社交媒体
- Instagram、LinkedIn、Reddit、TikTok、Twitter/X、YouTube 等

### 营销与邮件营销
- ActiveCampaign、Brevo、ConvertKit、Klaviyo、Mailchimp 等

### 支持与帮助台
- Freshdesk、Freshservice、Help Scout、Zendesk 等

### 电商与支付
- Shopify、Square、Stripe 等

### 设计与协作

### 分析与数据

## 使用方法

要使用这些技能，你需要：
1. 从 [platform.composio.dev](https://platform.composio.dev/) 获取 Composio API 密钥
2. 安装并配置 Composio 插件
3. 选择你想要自动化的应用技能
