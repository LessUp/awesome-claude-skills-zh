# 图像增强器

此技能获取你的图像和截图并使其看起来更好——更清晰、更专业。

## 何时使用此技能

- 为博客文章或文档提高截图质量
- 在社交媒体上分享之前增强图像
- 为演示文稿或报告准备图像
- 智能放大低分辨率图像
- 锐化模糊的照片
- 清理压缩图像

## 此技能的功能

1. **分析图像质量**：检查分辨率、锐度和压缩伪影
2. **增强分辨率**：智能放大图像
3. **提高锐度**：增强边缘和细节
4. **减少伪影**：清理压缩伪影和噪点
5. **优化用例**：根据预期用途调整（Web、打印、社交媒体）

## 提示

- 始终保留原始文件作为备份
- 最适合截图和数字图像
- 可以批量处理整个文件夹
- 如需指定输出格式（PNG 用于质量，JPG 用于更小文件）
- 对于社交媒体，提及平台以获得最佳尺寸

## 常见用例

- **博客文章**：发布前增强截图
- **文档**：使 UI 截图清晰可见
- **社交媒体**：为 Twitter、LinkedIn、Instagram 优化图像
- **演示文稿**：为大屏幕放大图像
- **打印材料**：为物理媒体增加分辨率
