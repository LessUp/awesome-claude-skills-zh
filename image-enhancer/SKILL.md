# 图像增强器

此技能处理你的图像和截图，使其看起来更好 - 更清晰、更锐利、更专业。

## 何时使用此技能

- 为博客文章或文档提高截图质量
- 在社交媒体分享前增强图像
- 为演示或报告准备图像
- 放大低分辨率图像
- 锐化模糊照片
- 清理压缩图像

## 此技能的功能

1. **分析图像质量**：检查分辨率、清晰度和压缩伪影
2. **增强分辨率**：智能放大图像
3. **提高清晰度**：增强边缘和细节
4. **减少伪影**：清理压缩伪影和噪点
5. **针对用例优化**：根据预期用途调整（Web、打印、社交媒体）

## 示例

**用户**："提高 screenshot-2024.png 的图像质量"

**输出**：
```
分析 screenshot-2024.png...

当前规格：
- 分辨率：1920x1080
- 格式：PNG
- 质量：良好，但略有模糊

应用的增强：
✓ 放大到 2560x1440（视网膜屏）
✓ 锐化边缘
✓ 增强文本清晰度
✓ 优化文件大小

保存为：screenshot-2024-enhanced.png
原始文件保留为：screenshot-2024-original.png
```

**灵感来源**：Lenny Rachitsky 的新闻通讯工作流程 - 用于文章中的截图

## 提示

- 始终保留原始文件作为备份
- 对截图和数字图像效果最佳
- 可以批处理整个文件夹
- 如需要可指定输出格式（PNG 用于质量，JPG 用于更小体积）
- 对于社交媒体，提及平台以获得最佳尺寸

## 常见用例

- **博客文章**：发布前增强截图
- **文档**：使 UI 截图清晰锐利
- **社交媒体**：为 Twitter、LinkedIn、Instagram 优化图像
- **演示**：为大屏幕放大图像
- **打印材料**：为物理媒介提高分辨率
