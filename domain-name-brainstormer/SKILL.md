# 域名头脑风暴器

此技能通过生成创意选项并检查实际可注册的域名，帮助你为项目找到完美域名。

## 何时使用此技能

- 启动新项目或公司
- 发布产品或服务
- 创建个人品牌或作品集网站
- 重新品牌现有项目
- 为副项目注册域名
- 当首选被占用时寻找可用替代方案

## 此技能的功能

1. **理解你的项目**：分析你在构建什么以及为谁构建
2. **生成创意名称**：创建相关、难忘的域名选项
3. **检查可用性**：验证哪些域名在多个 TLD 中实际可用
4. **多种扩展**：建议 .com、.io、.dev、.ai、.app 等
5. **提供替代方案**：如果首选被占用，提供变体
6. **品牌洞察**：解释为什么某些名称效果好

## 域名命名提示

### 什么是好域名

✓ **简短**：理想情况下 15 个字符以下
✓ **难忘**：易于回忆和拼写
✓ **可发音**：可以在对话中说
✓ **具描述性**：暗示你做什么
✓ **可品牌化**：足够独特以脱颖而出
✓ **无连字符**：更容易口口相传

### TLD 指南

- **.com**：通用、可信、适合企业
- **.io**：科技初创公司、开发者工具
- **.dev**：开发者聚焦产品
- **.ai**：AI/ML 产品
- **.app**：移动或 Web 应用
- **.co**：.com 的替代
- **.xyz**：现代、创意项目
- **.design**：创意/设计机构
- **.tech**：科技公司

## 示例工作流程

### 初创公司发布
1. 描述你的初创想法
2. 获得跨 TLD 的 10-15 个域名建议
3. 审查可用性和定价
4. 选择前 3 个最爱
5. 立即注册

### 个人品牌
1. 分享你的名字和职业
2. 获得变体（firstname.com、firstnamelastname.dev 等）
3. 也检查社交媒体账号可用性
4. 跨平台注册一致品牌

### 产品命名
1. 描述产品和目标市场
2. 获得创意、可品牌化名称
3. 检查商标冲突
4. 验证域名和社交媒体可用性
5. 与目标受众测试名称

## 成功提示

1. **快速行动**：好域名很快被注册
2. **注册变体**：获取 .com 和 .io 以保护品牌
3. **避免数字**：难以口口相传
4. **检查社交媒体**：确保 @username 也可用
5. **大声说出来**：测试是否易于发音
6. **检查商标**：确保无法律冲突
7. **考虑长远**：5 年后还有意义吗？

## 定价背景

建议域名时，我会注明：
- 标准域名：约 $10-15/年
- 高级 TLD（.io、.ai）：约 $30-50/年
- 已占用域名：如果列出的市场价格
- 高级域名：数百到数千美元
