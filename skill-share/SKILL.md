## 何时使用此技能

当你需要时使用此技能：
- **创建新的 Claude 技能**，带有适当的结构和元数据
- **生成技能包**，准备分发
- **自动在 Slack 频道上分享创建的技能**，提高团队可见性
- **验证技能结构**，在分享之前
- **打包和分发技能**给你的团队

也在以下情况使用此技能：
- **用户说他想要创建/分享他的技能**

此技能非常适合：
- 作为团队工作流的一部分创建技能
- 构建需要技能创建 + 团队通知的内部工具
- 自动化技能开发流水线
- 协作技能创建，带团队通知

## 关键功能

### 1. 技能创建
- 创建带有 SKILL.md 的适当结构的技能目录
- 生成标准化的 scripts/、references/ 和 assets/ 目录
- 自动生成带有所需元数据的 YAML frontmatter
- 强制执行命名约定（连字符大小写）

### 2. 技能验证
- 验证 SKILL.md 格式和必填字段
- 检查命名约定
- 在打包之前确保元数据完整性

### 3. 技能打包
- 创建可分发的 zip 文件
- 包含所有技能资产和文档
- 在打包之前自动运行验证

### 4. 通过 Rube 的 Slack 集成
- 自动将创建的技能信息发送到指定的 Slack 频道
- 分享技能元数据（名称、描述、链接）
- 发布技能摘要以供团队发现
- 提供技能文件的直接链接

## 工作原理

1. **初始化**：提供技能名称和描述
2. **创建**：创建带有适当结构的技能目录
3. **验证**：验证技能元数据的正确性
4. **打包**：将技能打包为可分发格式
5. **Slack 通知**：将技能详情发布到你团队的 Slack 频道

## 示例用法

```
当你要求 Claude 创建名为"pdf-analyzer"的技能时：
1. 创建 /skill-pdf-analyzer/ 并带有 SKILL.md 模板
2. 生成结构化目录（scripts/、references/、assets/）
3. 验证技能结构
4. 将技能打包为 zip 文件
5. 发布到 Slack："新技能已创建：pdf-analyzer - 高级 PDF 分析和提取功能"
```

## 与 Rube 集成

此技能利用 Rube 进行：
- **SLACK_SEND_MESSAGE**：将技能信息发布到团队频道
- **SLACK_POST_MESSAGE_WITH_BLOCKS**：分享丰富的格式化技能元数据
- **SLACK_FIND_CHANNELS**：发现技能公告的目标频道

## 要求

- 通过 Rube 的 Slack 工作区连接
- 技能创建目录的写入权限
- Python 3.7+ 用于技能创建脚本
- 用于技能通知的目标 Slack 频道
