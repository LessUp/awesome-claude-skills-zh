# 竞争对手广告提取器

此技能从广告库中提取竞争对手的广告并分析其有效之处——他们强调的问题、定位的用例以及产生共鸣的文案/创意。

## 何时使用此技能

- 研究竞争对手的广告策略
- 为你自己的广告寻找灵感
- 了解市场定位
- 识别成功的广告模式
- 分析有效的信息传递
- 发现新的用例或痛点
- 使用已验证的概念规划广告活动

## 此技能的功能

1. **提取广告**：从 Facebook 广告库、LinkedIn 等抓取广告
2. **截取屏幕截图**：保存所有广告的视觉副本
3. **分析信息传递**：识别问题、用例和价值主张
4. **分类广告**：按主题、受众或格式分组
5. **识别模式**：找到常见的成功方法
6. **提供洞察**：解释为什么某些广告可能表现良好

## 示例

**用户**："从 Facebook 广告库提取 Notion 的广告，告诉我哪些信息传递对他们有效。"

**流程**：
```
正在访问 Facebook 广告库...
搜索：Notion
找到：23 个活跃广告

正在截取屏幕截图...
[████████████████████] 100%

已保存至：competitor-ads/notion/
- ad-001-collaboration.png
- ad-002-productivity.png
- ad-003-templates.png
...

正在分析信息传递...
```

**输出**：
```
# Notion 广告分析

## 概览
- 广告总数：23 个活跃
- 主要主题：生产力（35%）、协作（30%）、
  模板（20%）、AI 功能（15%）
- 广告格式：静态图片（60%）、视频（40%）
- CTA 模式："免费试用"、"开始使用"

## 他们强调的关键问题

1. **信息分散**（8 个广告）
   文案："停止在 10 个不同工具之间切换"
   为什么有效：许多团队面临的直接痛点

2. **会议过载**（5 个广告）
   文案："用异步更新取代不必要的会议"
   为什么有效：COVID 后远程工作的痛点

3. **文档丢失**（4 个广告）
   文案："再也不要问'那个文档在哪里？'"
   为什么有效：普遍的职场挫折

## 成功的创意模式

### 模式 1：前后对比分割
- 展示混乱的工具景观 → 整洁的 Notion 工作区
- 用于 6 个高性能广告
- 视觉隐喻一目了然

### 模式 2：功能展示
- 实际产品使用的 GIF
- 在 5 秒内展示特定功能
- 用于新功能（AI、模板）

### 模式 3：社会证明
- "加入 2000 万用户"信息
- 客户 logo
- 用于 4 个针对企业的广告

## 有效的文案

最佳标题：
1. "你团队的知识，终于在一个地方"
   → 专注于利益，直接解决痛点
    
2. "多合一工作区"
   → 清晰的定位，广泛的吸引力
    
3. "真正帮助你工作的 AI"
   → 解决 AI 怀疑，实用角度

最佳正文文案模式：
- 短句（少于 10 个词）
- 关注结果而非功能
- 包含具体数字（"会议减少 50%"）

## 受众定位洞察

基于广告变体：
- 初创创始人：独立生产力角度
- 团队负责人：协作和协调
- 企业：安全性和合规性提及
- 学生：免费计划、模板、组织

## 对你广告的建议

1. **测试"工具蔓延"痛点**
   → 基于他们的广告频率，产生强烈共鸣

2. **使用产品截图而非抽象视觉**
   → 他们所有顶级广告都展示实际 UI

3. **以问题开头，而非解决方案**
   → "厌倦了 X？"比"推出 Y"表现更好

4. **文案保持在 100 个字符以内**
   → 他们最短的广告似乎最频繁

5. **测试前后对比视觉格式**
   → 他们创意中的已验证模式

## 已保存文件
- 所有广告：~/competitor-ads/notion/
- 分析：~/competitor-ads/notion/analysis.md
- 最佳表现：~/competitor-ads/notion/top-10/
```

## 最佳实践

### 法律与道德
✓ 仅用于研究和灵感
✓ 不要直接复制广告
✓ 尊重知识产权
✓ 使用洞察来启发原创创意
✗ 不要抄袭文案或窃取设计

### 分析技巧
1. **寻找模式**：什么主题重复出现？
2. **随时间追踪**：每月保存广告以查看演变
3. **测试假设**：为品牌调整成功模式
4. **按受众细分**：不同目标的不同信息
5. **比较平台**：LinkedIn 与 Facebook 信息传递不同

## 常见工作流

### 广告活动规划
1. 提取竞争对手广告
2. 识别成功模式
3. 注意他们信息传递中的空白
4. 头脑风暴独特角度
5. 起草测试广告变体

### 定位研究
1. 获取 5 个竞争对手的广告
2. 映射他们的定位
3. 找到服务不足的角度
4. 开发差异化信息
5. 与他们的方法对比测试

### 创意灵感
1. 按主题提取广告
2. 分析视觉模式
3. 注意颜色和布局趋势
4. 调整成功模式
5. 创建原创变体

## 成功技巧

1. **定期监控**：每月检查变化
2. **广泛研究**：也查看相邻竞争对手
3. **保存一切**：建立参考库
4. **测试洞察**：运行你自己的实验
5. **追踪表现**：A/B 测试启发概念
6. **保持原创**：用于灵感，而非复制
7. **多平台**：比较 Facebook、LinkedIn、TikTok 等

## 输出格式

- **屏幕截图**：所有广告保存为图片
- **分析报告**：洞察的 Markdown 摘要
- **电子表格**：包含广告文案、CTA、主题的 CSV
- **演示文稿**：最佳表现者的视觉幻灯片
- **模式库**：按方法分类
